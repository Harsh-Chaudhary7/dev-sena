# Dev-Sena 🚀

Dev-Sena is a smart developer companion for VS Code that helps you stay productive while coding.

It combines:

- ⚡ Code Speedometer (live typing speed)
- 🧠 AI-ish Log Watcher
- 🛠 Error Explainer

All inside your editor — no extra tools needed.

---

## ✨ Features

### ⚡ Code Speedometer
- Shows real-time typing speed (CPM – characters per minute)
- Ignores backspace/delete for accurate tracking
- Automatically resets after 20 seconds of inactivity

Example:
⚡ Speed: 320 CPM
